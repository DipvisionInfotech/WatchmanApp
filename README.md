# Watchman - Secure Gate Management App

The Watchman App is designed for security guards to efficiently manage visitor entries, exits, deliveries, domestic staff, and vehicle access within residential communities, apartments, and gated societies.

## Key Features

*   **🔐 Secure Login:** Mobile Number + OTP or Username/Password.
*   **👤 Visitor Management:** Register visitors, capture photos, and send approval requests to residents.
*   **✅ Resident Approval:** Real-time approval/rejection from residents.
*   **📦 Delivery Management:** Track delivery executives (Food, Courier, etc.).
*   **👷 Staff Management:** Daily check-in/out for maids, cooks, and drivers.
*   **🚗 Vehicle Entry:** Log vehicle numbers and owner details.
*   **🚨 Emergency Alerts:** Resident SOS alerts and quick emergency contact access.
*   **📊 Daily Reports:** Summary of visitors, deliveries, and attendance.

## Tech Stack

*   **Framework:** Flutter
*   **State Management:** Provider
*   **Design:** Material 3 with Google Fonts (Poppins)
*   **Icons:** Material Icons & Font Awesome

## Getting Started

1. Clone the repository.
2. Run `flutter pub get`.
3. Run the app using `flutter run`.
